# Annotation Agent MVP Roadmap

## Goal

Build a beginner-friendly annotation management MVP for text classification:

1. Create a project
2. Configure labels
3. Import text items
4. Run AI pre-labeling
5. Confirm labels manually
6. Run quality check
7. View basic metrics

## Phase 1

Set up the backend project structure and database models.

## Backend Structure

- `backend/app/api/`: API routes
- `backend/app/core/`: settings and shared config
- `backend/app/db/`: database setup
- `backend/app/models/`: database models
- `backend/app/schemas/`: request/response schemas
- `backend/app/services/`: business logic and agent logic
